
# Real-Time-Chat

Real-Time-Chat is a full-stack real-time chat application built using the MERN stack. 
It supports real-time messaging using Socket.IO and securely stores user data in MongoDB with encrypted credentials.
## Tech Stack

**Client:** React.js

**Server:** Node.js, Express.js

**Database:** MongoDB

**Authentication:** JWT (JSON Web Tokens) with bcrypt  

---

## Run Locally

Go to the project directory

```bash
  cd RealTime_Chat-App
```

Install dependencies

```bash
  cd backend/
  npm install
```

```bash
  cd frontend/
  npm install   
```

Start the server

```bash
 //open now terminal
 cd backend
  npm run dev
```
Start the Client

```bash
  //open now terminal
  cd frontend
  npm run dev
```
